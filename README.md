# NaverPay
## 설명
- 스프링의 MVC구조의 이해를 목적으로 제작한 네이버페이 웹 기능 구현 프로젝트입니다.

## 개발환경
- Tomcat 9.0.68
- Java 11 
- Spring 5.3.23

## 역할
- 차승준, 이미지 (1팀) : 메인 페이지, 로그인 페이지, 로그인 시도
- 유재학, 이규석, 이강훈 (2팀) : 구매 내역 페이지, 구매 내역 리스트 반환, 특정 기간 동안의 구매 내역 리스트 반환
- 최병옥, 김예지 (3팀) : 구매 상세 정보 페이지, 구매 상세 정보 리스트 반환, 구매 상세 정보 삭제
